\section{Custom Commands}

Example using physics2 package's \verb|\ab()| command:

\[
\ab(x + y) = \left( x + y \right)
\]

Nested bracket example:

\[
\ab(\ab(a) + b) = \text{This kind of expression is also supported}
\]
